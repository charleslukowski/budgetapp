## User Workflows

### Budget/Forecast Input (replaces Excel macro workflow)
1. Navigate to Budget Input
2. Select plant (Kyger/Clifty) and scenario (2025 Forecast)
3. Click on department to edit (e.g., MAINT, PLANNED-01)
4. Add/edit line items with monthly amounts
5. Mark department as "Ready" when done
6. Repeat for all departments
7. Submit to database (replaces Excel macro)
8. Review Combined view to verify all data

### Monthly Close Process
1. Import GL actuals (Energy + Expense)
2. Import coal data from Aligne
3. Review Department Summary (Fcst vs Actual)
4. Review variance dashboard
5. Add variance explanations
6. Generate sponsor report
7. Download and distribute

### Forecast Update (Mid-Year Revision)
1. Navigate to Budget Input
2. Select scenario to update
3. Edit department line items as needed
4. Update fuel drivers (generation, prices)
5. Review Department Summary
6. Compare to budget
7. Submit revised forecast

### Sponsor Report Generation
1. Navigate to Reports
2. Select report type and period
3. Choose plant(s)
4. Generate report
5. Download Excel file
6. (Optional) Email to sponsors

## Key Design Decisions

### Multi-User Editing
- Multiple users can edit different departments simultaneously
- Department-level locking when someone is editing
- Show "Currently editing: [username]" indicator
- Auto-save with conflict detection

### Budget vs Forecast Distinction
| Type | Frequency | Seeding | User Action |
|------|-----------|---------|-------------|
| Budget | Annual | Seed with account/description list | Fill amounts, add new rows |
| Forecast | Monthly | Copy from current budget | Update amounts, add new rows |
