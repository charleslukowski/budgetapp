# Fuel Model UI Wireframe Concepts

## Problem Statement

Current input structure mirrors Excel's linear, cell-by-cell approach:
- 12 monthly use factor inputs
- Multiple individual driver fields
- Results shown separately after calculation

**Goal**: Create an intuitive interface that surfaces the key drivers while hiding complexity, enables quick scenario exploration, and shows live feedback.

---
